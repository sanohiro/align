# 8. Whose Is It?

> 🌐 [English](../08-whose-is-it.md) · **日本語**

**Q1.**

```align
a := 42
b := a
```

いま 42 はいくつ存在しますか?

**A1.** 2つ。数は **Copy** です。代入は複製します。それぞれが自分の道を行きます。

---

**Q2.**

```align
a := "hello".clone()    // an owned string
b := a
```

いま所有された string はいくつ存在しますか?

**A2.** 1つ。`string` はヒープ上のバッファを所有し、所有者は **Move** です。バッファが *持ち主を替えた* のです。`b` がそれを持っています。

---

**Q3.** では、その直後の `print(a.len())` は何をしますか?

**A3.** コンパイルできません: *ムーブ済みの値 `a` の使用*。古い名前は死んでいます。1つのバッファ、1人の所有者、例外なし。

---

**Q4.** なぜそんなに厳しいのですか?

**A4.** 所有者が1人なら、コンパイラは誰がいつ解放するかを正確に知れるからです。待つべきガベージコレクタもなく、恐れるべき二重解放もなく、書くべきライフタイム注釈もない。この厳しさ *こそが* メモリ管理なのです。

---

**Q5.** でも、本当に string が2つ欲しいのです。

**A5.** なら、本当に代価を払ってください: `b := a.clone()`。ディープコピーを、コストのある場所ではっきり綴って。Align では何でもコピーできます — ただ、*見えないところで* コピーすることだけができないのです。

---

**Q6.** どの型が Move ですか?

**A6.** 所有者たち: `string`、`array<T>`、`buffer`、`box`、ファイルハンドル。素朴なものはすべて — 数、`bool`、view、それらの小さな構造体 — Copy です。リソースを所有するなら Move。ただのデータなら Copy。そして `string` を *含む* 構造体もまた Move になります — 所有権は上へと染み上がるのです。

---

**Q7.** どんな `.clone()` もする前の `"hello"` は何ですか — このリテラルは誰が所有していますか?

**A7.** 誰も。これは `str` — **view** です。ポインタと長さで、自分より長生きするバイト列を眺めている。view は Copy です — データへの *眺め* をコピーするのはタダですから。

---

**Q8.** さて arena です。これは何を表示しますか?

```align
fn shout(name: str) -> string {
    arena {
        s := "hey, " + name + "!"
        return s.clone()
    }
}

print(shout("align"))
```

**A8.** `hey, align!`。`arena { }` の中では、`+` の連結が自由に確保します — arena の中へ。`}` で、arena まるごとがひとつの動作で解放されます。`.clone()` が、その前に生き残りを外へコピーしておきました。

---

**Q9.** clone なしで `return s` としたら?

**A9.** コンパイルできません: *arena で確保した値は返せません*。コンパイラは `s` のリージョンと、それが死ぬ `}` を知っています。脱出はコピーであり、コピーは目に見えるもの: `.clone()`。

---

**Q10.** arena に手を伸ばすのはいつですか?

**A10.** ある *フェーズ* が、ともに死ぬ多くのものを確保するとき — このファイルを parse する、このリクエストを処理する、このバッチを decode する。フェーズを `arena {}` でひとくくりにすれば、一時領域はそれぞれポインタを1つ進めるだけのコスト。後始末は1行きり、そして忘れようがありません。

---

**Q11.** では、あらゆる新しいデータについての、決定まるごとは?

**A11.** 問いは一つ — *それはどれだけ生きるのか?*

- このスコープ → 素朴な値、それで終わり
- このフェーズ → arena、生き残りは `.clone()`
- もっと長く、所有者1人 → 所有型、move して連れていく
- ただ眺めているだけ → view、タダ

---

**Q12.** では、いったい誰のものですか — この行の、まさにこのバッファは?

**A12.** ソースを読んでください: それが最後に move した先の束縛です。Align の所有権は実行時の謎ではありません。書き下されていて、しかもあなたの読みはコンパイラがすでに確かめてくれています。

---

> **第八の戒律**
>
> *所有者は一度に1人。短命なものは arena のもとにまとめよ。そしてどうしても2つ要るなら — `.clone()`、皆に見える場所で。*
