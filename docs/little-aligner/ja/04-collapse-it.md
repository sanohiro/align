# 4. Collapse It

> 🌐 [English](../04-collapse-it.md) · **日本語**

**Q1.** `sum` と `count` にはもう出会いました。`[5, 3, 8, 1].min()` は何ですか?

**A1.** `1`。

---

**Q2.** そして `[5, 3, 8, 1].max()` は?

**A2.** `8`。ここまでの reducer は `sum`、`count`、`min`、`max`。どれも多くを一つに畳み込みます。

---

**Q3.** `[1, 2, 3, 4].any(fn x { x > 3 })` は何ですか?

**A3.** `true` — 少なくとも一つの要素が通ります。兄弟分の `all(fn x { x > 0 })` も、ここでは `true`。`any` と `all` は `bool` に畳み込まれます。

---

**Q4.** `[2, 4, 6].all(fn x { x % 2 == 0 })` は何ですか?

**A4.** `true`。では `[2, 4, 7]` は? — `false`。`7` が台無しにします。そして `all` は、それに出会った瞬間にそう言えるのです。

---

**Q5.** ぴったりの reducer がないとしましょう。今度は *積* が欲しい。どうしますか?

**A5.** 万能のもの: `reduce`。

```align
[1, 2, 3, 4].reduce(1, fn acc, x { acc * x })
```

`24`。

---

**Q6.** なぜ先頭に `1` があるのですか?

**A6.** 初期値です — まず種、それから畳み込む関数。`acc` は `1` から始まり、各要素が掛け込まれます。和ならば `0` を種にするでしょう。種は、空の配列が返す答えでもあります。

---

**Q7.** `fn acc, x { acc * x }` で、どちらがどちらですか?

**A7.** `acc` はこれまでの答え、`x` は次の要素。lambda は、次の要素がどう答えに加わるかを言っています。

---

**Q8.** `reduce` を使って、`sum` を自分で書いてください。

**A8.** `xs.reduce(0, fn acc, x { acc + x })`。これで分かりました — `sum`、`count`、`min`、`max` は気遣い(近道)にすぎず、`reduce` こそが万能のエンジンで、名前つきのものは意図をより手早く言うためのものなのです。

---

**Q9.** `[1, 2, 3, 4].scan(0, fn acc, x { acc + x })` は何ですか?

**A9.** *途中経過* の和: `1, 3, 6, 10`。`scan` は途中の仕事を見せる `reduce` です — だからそれは**ステージ**であって、終わりではありません。まだ何かがそれを消費しなければなりません。

---

**Q10.** では `[1, 2, 3, 4].scan(0, fn acc, x { acc + x }).max()` は何ですか?

**A10.** `10` — 途中経過の和が到達した最大値。(配列に負の数があると、この問いは面白くなります。いわば最高水位標です。)

---

**Q11.** `[3, 1, 2].map(fn x { x }).sort()` は何ですか — 順序を頼んでもいいですか?

**A11.** `[1, 2, 3]`、実体化されて。`sort` は並べ替えるために配列を作らねばなりません — だから `to_array` と同じく、本物の配列を手渡します。確保は白日のもとに。

---

**Q12.** そして `[10, 21, 32, 3].sort_by_key(fn x { -x })` は?

**A12.** `[32, 21, 10, 3]`。符号を反転した値で並べ替える: `reverse` を一切使わずに降順です。

---

**Q13.** `[1, 2, 3, 4, 5].partition(fn x { x % 2 == 0 })` は何ですか?

**A13.** 一度に2つの配列: `([2, 4], [1, 3, 5])` — 認められたもの、それから残り。まとめて受け取りましょう: `(evens, odds) := ...`。

---

> **第四の戒律**
>
> *まず種、それから畳め。そして名前つきの reducer のほうがはっきり言えるなら — `sum`、`count`、`min`、`max`、`any`、`all` — はっきり言え。*
