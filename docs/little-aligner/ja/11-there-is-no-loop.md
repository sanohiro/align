# 11. There Is No Loop

> 🌐 [English](../11-there-is-no-loop.md) · **日本語**

**Q1.** Align の `for` はどこにありますか?

**A1.** ありません。`while` もありません。あなたは10章のあいだ、それに気づかず暮らしてきました — pipeline がループだったのです。

---

**Q2.** でも、本当に一歩ずつ進む処理もあります。3 から 1 までカウントダウンするには?

**A2.** 自分自身を呼ぶ関数:

```align
fn countdown(n: i64) -> i64 {
    if n == 0 { return 0 }
    print(n)
    return countdown(n - 1)
}
```

`3, 2, 1`。再帰が、順次処理のための道具です。

---

**Q3.** そういう関数がどれも持つ、2つの部分は何ですか?

**A3.** **基底ケース**(`n == 0` — 止まる)と **ステップ**(少し仕事をして、より小さい何かで再帰する)。基底が先。いつでも基底が先。

---

**Q4.** 1 から n までの和を、再帰で?

**A4.**

```align
fn sum_to(n: i64, acc: i64) -> i64 {
    if n == 0 { return acc }
    return sum_to(n - 1, acc + n)
}
```

`sum_to(10, 0)` は `55`。

---

**Q5.** そこで `acc` は何をしていますか?

**A5.** これまでの答えを、呼び出しの *下へ* 運んでいます。だから最後の呼び出しは、もう結果を手にしている。これは `reduce` のアキュムレータを手で巻いたもの — 鏡に映った4章です。

---

**Q6.** 1000ステップ — 1000個のスタックフレーム?

**A6.** いいえ。`return sum_to(...)` は **末尾呼び出し(tail call)** です — 再帰をそのまま返し、呼び出しのあとにすることは何も残らない — そしてそれはジャンプにコンパイルされます。同じフレームが再利用されるのです。再帰を末尾位置に保てば、深さはタダ。

---

**Q7.** ではこちらは — 末尾呼び出しですか、そうでないですか?

```align
fn sum_to(n: i64) -> i64 {
    if n == 0 { return 0 }
    return n + sum_to(n - 1)
}
```

**A7.** 違います — 呼び出しが返ったあとに `n +` がまだ待っています。だからフレームは生き続けねばなりません。正しいけれど、積み上がる。アキュムレータ版は `+` を呼び出しの *前* にやります。それが Q4 の手品のすべてです。

---

**Q8.** ファイルを4096バイトずつ、空になるまで読みます。ループの形 — つまり、再帰の形?

**A8.**

```align
fn pump(r: reader, w: writer, buf: buffer) -> Result<(), Error> {
    n := r.read(buf)?
    if n == 0 { return Ok(()) }
    w.write(buf.bytes())?
    return pump(r, w, buf)
}
```

基底ケース: EOF。ステップ: ひと飲み。末尾呼び出し: 同じバッファ、次の回。これが標準ライブラリの、お手本どおりの I/O ループです。

---

**Q9.** Q8 のように、`?` と再帰は関数を共有できますか?

**A9.** たったいま、そうしました。どんな失敗も、下降の *まるごと* から一度に `Err` を返します — 「ループ」からの早期脱出を、`break` の仕掛けなしで。

---

**Q10.** いつ再帰し、いつ pipeline にするのか? 最後の組分け帽子です:

**A10.**

- 多くの要素に同じ行いを → **pipeline**(2〜5章)。
- グループ畳み込み → **`group_by`**(10章)。
- *各ステップが、前のステップの作ったものを必要とする* — parser、状態機械、空になるまでひと飲みずつ → **再帰**、アキュムレータを手に、呼び出しは末尾で。

---

**Q11.** あなたは `i` を `0` から `len` まで再帰させ、毎回 `xs[i]` に同じことをしています。何をしましたか?

**A11.** 変な帽子をかぶった `for` ループを書きました。帽子を脱いでください — それは最初から `xs.map(...)` だったのです。

---

> **第十一の戒律**
>
> *基底ケースを先に、アキュムレータを手に、呼び出しは末尾で。そしてそもそも再帰する前に — それが pipeline ではなかったか、問え。*

---

**Q12.** これで終わりですか?

**A12.** ドリルは、はい。Align は — [ガイド](../../guide/ja/README.md)に残りがあります: string、JSON、並列性、SIMD、標準ライブラリ。でもあなたはもう、Align を外国語のようには読みません。メニューのように読むのです。

*さあ、何か作りにいきましょう。*
